# docker-compose-for-home

## Instructions

This repository is a compilation of docker-compose files to differents services that you can deploy in your home.

You must sustitute \<variable-name> by your own parameters.

## List

* Duck DNS Script
* Heimdall
* NextCloud
* OpenVPN
* Pi-hole
* Pigallery
* Plex server
* Samba server
* Serviio
* Transmission
